- `Programming Contest Hosting Guide`는 학교 교내 프로그래밍 경진대회나 개인이 주최하는 프로그래밍 경진대회를 운영하는 분들께 도움을 주고자 만든 가이드 및 체크리스트를 제공하는 프로젝트입니다.
- 본 프로젝트는 다양한 분들께서 제공해주신 메뉴얼과 그리고 조언들 및 제가 경험했던 내용들을 토대로 작성하였으며, 2026년 4월 28일 Baekjoon Online Judge가 서비스 종료됨에 따라 여러 OJ들에서의 가이드들을 최대한 담고자 하였습니다.
	- `도움 주신 분들` 페이지를 참고해주시기 바랍니다.
- 아직 이 프로젝트는 미완성입니다. PR등 다양한 기여는 언제든지 환영합니다.
